操作接口。

注解
  • 此接口仅由是用户使用。它由插件实现。
  • 它的实例在主线程（界面线程）中。

  • 控制台类： COperateTerminal
  • 文件传输： COperateFileTransfer
  • 服务类： COperateService

◆ sigRunning

void COperate::sigRunning ( )
signal

开始成功信号。仅由插件触发。

  • 应用程序需要在此时设置视图属性 Enabled 为 true 。允许接收键盘和鼠标事件。
  • 如果插件自己实现视图（非　CFrmView），设置视图属性 Enabled 为 true 。允许接收键盘和鼠标事件。
