简介

**玉兔远程控制**是一个开源、跨平台、支持多协议的远程控制软件。

能让你在**任何地方**使用**任何设备和系统**通过**任意方式**远程管理**任意设备和系统**。

它的目标是简单、方便、易用、安全、提高工作效率。

它包括远程桌面、远程控制、文件传输（FTP、SFTP）、终端、远程终端（SSH、TELNET）、播放器、网络工具等功能。

支持协议

远程控制协议

  • [x] VNC（虚拟网络计算）也叫 RFB(远程帧缓存): https://github.com/rfbproto/rfbproto
  • [x] RDP（远程桌面协议）: https://github.com/FreeRDP/FreeRDP/wiki/Reference-Documentation
  • [ ] Spice: https://www.spice-space.org/
  • [x] 终端
  • [x] SSH: https://www.rfc-editor.org/info/rfc4251
    • [x] libssh
  • [x] TELNET: https://www.rfc-editor.org/info/rfc318
  • [ ] scrcpy: https://github.com/Genymobile/scrcpy
  • [x] FTP（文件传输协议）: https://www.rfc-editor.org/info/rfc959
  • [x] SFTP(安全 Shell 文件传输协议)：https://tools.ietf.org/html/draft-ietf-secsh-filexfer-02

网络协议

  • [x] 端到端（P2P）协议
    • ICE（交互式连接建立）: RFC 8445
    • STUN: RFC 5389
    • TURN(在 NAT 周围使用中继进行遍历): RFC 5766
  • [x] 代理
    • [x] socks
    • [ ] http
    • [x] 端到端（P2P）代理协议： https://github.com/KangLin/RabbitRemoteControl/issues/7
