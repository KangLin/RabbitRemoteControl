Detailed Description

Service library.

  • Threading model: A plugin starts a thread to process a service. See the specific implementation: CPluginServiceThread and CService
  • Class relationship
  • usge:
    • Implement plugin interface:
      • CPluginService: Plugin interface
      • CPluginServiceThread: The default start a new thread. eg. CPluginServiceRabbitVNC
        • Plugin name: This name must be the same as the project name (${PROJECT_NAME}). The translation file (${PROJECT_NAME}_*.ts)) name is associated with it. E.g. CPluginServiceRabbitVNC::Name()
    • Implement CService . Eg. CServiceRabbitVNC
