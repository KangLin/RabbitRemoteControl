Detailed Description

Parameter interface.

  • The parameter has the following type:
    1. Only valid in the plugin. The application cannot access it, but the application can be set it via COperate::OpenDialogSettings.
      See also
      CParameterOperate category CParameterOperate CParameterBase
    2. Valid in the CManager and the plugin.
      The application cannot access it, but the application can be set it via CManager::GetSettingsWidgets.
      CManager and Plugins can use them directly. The plugin maybe use them as initialization values. Use CParameterPlugin
      See also
      CParameterClient COperate::SetGlobalParameters
    3. Valid in both the application and the Client and the plugin. This type is not used at this time.
