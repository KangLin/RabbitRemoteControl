Detailed Description

Parameter interface.

  • The parameter has the following type:
    1. Only valid in the plugin. The application cannot access it, but the application can be set it via CConnecter::OpenDialogSettings.
      See also
      CParameterConnecter category CParameterConnecter CParameterBase
    2. Valid in the CClient and the plugin.
      The application cannot access it, but the application can be set it via CClient::GetSettingsWidgets.
      CClient and Plugins can use them directly. The plugin maybe use them as initialization values. Use CParameterClient
      See also
      CParameterClient CConnecter::SetParameterClient CParameterUser::OnLoad
    3. Valid in both the application and the Client and the plugin. This type is not used at this time.
