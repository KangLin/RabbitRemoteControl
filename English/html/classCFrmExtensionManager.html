Detailed Description

扩展管理器界面

  • Qt 版本要求：必须使用 Qt 6.10 或更高版本
  • 扩展格式：只支持 Manifest V3 格式的扩展
  • 配置文件：不能使用 off-the-record 的配置文件

查找 Chrome 已安装的扩展：

  • 在 Chrmoe 地址栏中输入： chrome://version/
  • 在其中找到“个人资料路径”
  • Chrome 扩展位于“个人资料路径”下的“Extensions”目录下
